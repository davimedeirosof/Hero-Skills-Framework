# Skills — Índice

211 skills instaladas via Hero-Skills-Framework.
Localizadas em `~/.claude/skills/`.

## Como usar

Skills são ativadas automaticamente pelo Orquestrador nos pipelines Opensquad,
ou podem ser referenciadas diretamente em comandos.

## Categorias principais

| Categoria       | Skills                                                                 |
|-----------------|------------------------------------------------------------------------|
| Desenvolvimento | ln-400 a ln-500 (executores de tasks e stories)                       |
| Qualidade       | ln-510 a ln-640 (auditores de código, testes, documentação)           |
| Performance     | ln-810 a ln-830 (otimização, dependências, modernização)              |
| Infraestrutura  | ln-730 a ln-780 (docker, CI/CD, segurança, healthcheck)               |
| Marketing       | mktgskill-* (copywriting, SEO, email, ads, social, CRO)               |
| Documentação    | ln-100 a ln-160 (criação e auditoria de docs por camada)              |
| Orquestração    | ln-200 a ln-310 (decomposição, épicos, stories, tasks)                |
